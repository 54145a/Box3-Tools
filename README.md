# Box3-Tools
Nomen提供的基础工具以及四处收集的Box3框架上发展出的代码，在文件中会按照格式标定来源，运用，以及是否为模块

提示：你所看到的文档开头应该是这样的：
```
/**
 * !info {Project} -来自XXX
 * 一些描述
 */
```

或

```
/**
 * !info {Module} -来自XXX
 * 一些描述
 */
```

这些元素所指示的是该文件的元元素
Project代表这是一个实例代码，Module表示这是一个模块代码
模块代码可以直接导入，作为工具使用，实例代码则用于参考
